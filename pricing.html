04FAQ
Straight answers
Why are paid tiers needed if Forge uses my own AI keys?

Free tier covers the most-used BUILD tools (resume + bullets + portfolio + gap + achievement). The other 12 tools — ATS analysis, tailoring, outreach, strategy, interview prep — are where the real research, prompt engineering, and ongoing tuning live. Paid tiers also remove the watermark, unlock cloud sync, designer templates, and priority routing across providers when one rate-limits.

What payment methods do you accept?

UPI, all major Indian cards, netbanking, wallets — via Razorpay. INR only at launch; Stripe-powered global payments coming Phase 2.

Can I cancel anytime?

Yes. Monthly cancels at the end of the cycle. Day pass auto-expires after 24h. Yearly refunded pro-rata if you cancel within 14 days.

What about my AI keys?

Forge uses bring-your-own-key. You generate free keys at Groq, Gemini, OpenRouter (60 seconds each, no card). You paste them into Forge's ▸ KEYS panel. You can revoke any key from your provider's dashboard whenever you want.

Do you offer refunds?

Day pass: not refundable (24h product). Monthly: full refund within 7 days, no questions. Yearly: pro-rata within 14 days.

Is there an API for power users?

API access is on the roadmap for the YEARLY tier (and Enterprise). See API & Documentation.
