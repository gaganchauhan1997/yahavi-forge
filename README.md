| # | Module | What it forges |
|---|--------|----------------|
| 01 | **AI Resume Builder** | Raw experience → STAR-method achievement bullets in 5 tones (Corporate, Startup, FAANG, Creative, Executive) |

## ▸ RUN LOCALLY

```bash
git clone https://github.com/gaganchauhan1997/yahavi-forge.git
cd yahavi-forge
python3 -m http.server 8000
```

Open `http://localhost:8000` and click `▸ KEYS` to add your free API key.

That's it. No build step. No dependencies. No backend.
